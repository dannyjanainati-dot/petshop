# 🐾 Sistema PetShop

## 📋 Sobre o Projeto

O **Sistema PetShop** é um programa desenvolvido em Python com o objetivo de auxiliar no gerenciamento de informações de um pet shop. O sistema permite cadastrar, buscar, listar e inserir dados de forma simples e organizada.

O projeto foi criado para facilitar o controle de informações armazenadas no banco de dados, tornando o gerenciamento mais prático e eficiente.

## 🎯 Objetivos

* Cadastrar novos dados no sistema;
* Buscar informações já registradas;
* Listar todos os dados cadastrados;
* Organizar e gerenciar as informações do pet shop.

## ⚙️ Funcionalidades

✅ Cadastro de novos registros.

✅ Busca de informações no banco de dados.

✅ Listagem de todos os registros cadastrados.

✅ Inserção e armazenamento de dados de forma organizada.

## 🛠️ Tecnologias Utilizadas

* Python
* Banco de dados SQLite (`petshop.db`)

## 🚀 Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/petshop.git
```

2. Entre na pasta do projeto:

```bash
cd petshop
```

3. Execute o programa:

```bash
python petshop.py
```

ou, dependendo da versão instalada:

```bash
python3 petshop.py
```

## 💾 Banco de Dados

O arquivo `petshop.db` é responsável por armazenar as informações cadastradas no sistema, permitindo que os dados sejam mantidos mesmo após o encerramento do programa.
